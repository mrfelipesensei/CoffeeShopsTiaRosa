**Coffee Shop - Front-End** ☕ **Status:** Concluído - Projeto Acadêmico

Uma interface moderna e responsiva desenvolvida para uma cafeteria fictícia. O foco principal deste projeto foi a applicação de conceitos de **Design Responsivo**, **Semântica HTML** e **Estilização Avançada**.

🎨 **Destaques do Projeto**
- **Design Adaptável:** Totalmente responsivo para Mobile, Tablet e Desktop;
- **Interatividade:** Menu de navegação suave e efeitos de hover em cards e botões;
- **Fidelidade Visual:** Implementação rigorosa de paleta de cores e tipografia voltadas para o nicho gastronômico.

🛠 **Tecnologias**
- **Estrutura:** HTML5 Semântico
- **Estilização:** CSS3 - Flexbox, Grid e MediaQuery;
- **Interações:** JavaScript puro.
